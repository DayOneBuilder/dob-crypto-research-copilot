# DOB Crypto Copilot Starter

This starter bundle gives you three repeatable crypto research workflows that are easier to run inside an agent than a pile of random notes.

Use it when you want a cleaner process for:

- checking whether a token deserves more time
- reviewing a wallet without losing the thread
- comparing yields without hopping between tabs blindly

## Included workflows

### 1) token-dd
Use this when you want a fast but disciplined first pass on a token, ticker, or contract.

### 2) wallet-check
Use this when you want a summary of what a wallet holds, how it behaves, and what looks unusual.

### 3) yield-scan
Use this when you want to compare opportunities by chain, protocol, TVL, reward source, and obvious risks.

## How to use the bundle

Open the workflow file that matches the job. Paste the workflow into your preferred agent, then replace the placeholders with the asset, address, chain, or niche you care about.

Treat this as a working structure, not as blind automation. The output is only useful if the data and reasoning still make sense when you pressure-test them.
